# Advanced Consulting — Wuthering Waves

## Diagnose in order

1. **Rotation mechanics**: Concerto handoff, Intro/Outro target, Forte loop, Resonance Skill and Liberation timing, Echo active timing, swap cancels, Tune Break.
2. **Energy and field time**: enough Energy Regen for the intended cycle, no overlong support field time, no missed Outro.
3. **Investment floor**: resonator/weapon levels, key skill levels, inherent skills, sequence breakpoints.
4. **Echo structure**: correct cost pattern, Sonata set, main stats, tuned substats, active Echo.
5. **Team fit**: buffs/debuffs, sustain, grouping, chamber demands, enemy resistance, stagger/Tune Break needs.
6. **Material plan**: resonator ascension, weapon ascension, Forte/skill upgrade materials, boss drops, weekly limits, and whether the user should farm now or wait.
7. **Account planning**: waveplate efficiency, weekly bosses, echo routes, event rewards, pull opportunity cost.

## Rich-text structure

- Prefer compact image-card Markdown tables for grouped recommendations. Keep long explanations below the table as short labeled lines.
- Echo advice must include both active Echo and Sonata set when relevant, with local icons if cached.
- Material advice must show the key farm items with local material icons when cached.
- End source lists under `분석글`, not `근거:`.

## Pull advice policy

- Always ask for or infer from screenshots: pity, guarantee status, Astrite/Tide count, roster gaps, weapon ownership, and spending boundary.
- Frame recommendations as `가치`, `대체 가능성`, `리스크`, `기회비용`.
- Never imply a paid pull is mandatory. Recommend waiting when current evidence is insufficient.

## Endgame framing

- Tower/Endstate checks should identify team split, sustain needs, burst windows, and whether a second-side roster bottleneck matters more than the featured banner.
- If the user asks for exact clears or speedrun optimization, current chamber data and matchup-specific guides are required.
