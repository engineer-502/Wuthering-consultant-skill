# Report Design

Visual report mode should produce dark, clean, Wuthering Waves-inspired cards:

- Background: deep navy/black gradient with cyan/teal accents.
- Sections: conclusion, build state, echo/Sonata plan, weapon ranking, material plan, team rotation, source notes.
- Do not print long citations inside cards; keep citations in text answer.
- Use official/wiki-first images or local cache. If missing, use generated placeholders and mark image gaps.
- Korean labels by default.
- Keep source notes out of dense tables. In the text answer, list them vertically under `분석글`.
- Material plan cards should use local `material` icons for ascension, weapon, Forte/skill, EXP, and boss-drop items when available.
